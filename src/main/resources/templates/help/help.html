Table Options

The search bar will search on all columns.
cached This icon will reload the data cached in the table.
apps This icon will show which columns to display.
create This icon in the edit column brings up the UI to create or edit a Nucing Report.

The calandar will automatically be filled with events based on when the nucs are made in the Nuc Report

On the right you will see statistics that come from the sum of all the nuc reports. Below the statistics are 2 special functions:

  • add_circle_outline This button adds a new nuc yard into the database.
  • create This button edits an existing nuc yard. This is how a yard can be deactivated and reactivated.
